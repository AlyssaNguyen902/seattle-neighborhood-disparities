## Data

This folder contains the analysis-ready dataset used for the Seattle Neighborhood Disparities project.

### Dataset
- `seattle_neighborhood_analysis.csv`

### Description
Neighborhood-level socioeconomic, demographic, and health indicators for Seattle. The dataset is a cleaned and reduced version of multiple publicly available sources and was created specifically for analysis and visualization.

### Unit of Analysis
- Seattle neighborhood

### Data Sources
- Racial and Social Equity Index — Seattle  
- Disability and Health Insurance — Seattle Neighborhoods  

### Notes
- Only variables used in the analysis and Shiny application are included
- Original raw datasets and intermediate joined files are not included in this repository
- All data reflect the post–Affordable Care Act period
